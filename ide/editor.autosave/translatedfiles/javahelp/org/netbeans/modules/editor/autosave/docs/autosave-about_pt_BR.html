Sobre Salvar automaticamente

Este módulo provê a funcionalidade de salvar automaticamente todos os arquivos não salvos abertos.

É possível ainda definir se este recurso estará habilitado ou não, bem como o intervalo de tempo entre as operações de salvar. Para alterar estas opções, favor utilizar as opções em Ferramentas > Opções > Editor > Salvar automaticamente. Para habilitar/desabilitar este recurso, é possível utilizar-se ainda a opção Arquivo > Salvar automaticamente.
